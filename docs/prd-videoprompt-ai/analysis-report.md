## 3. 竞品分析

### 竞品列表

| 产品名称 | 核心功能 | 定价模式 | 差异点 | 市场表现 |
|---------|---------|---------|--------|---------|
| PromptHero | AI 提示词搜索引擎，涵盖图像/视频/文本提示词 | 免费浏览 + Pro 订阅 | 专注搜索和分享，非编辑器；无结构化参数 | 百万级月访问 |
| PromptBase | AI 提示词交易市场，用户买卖高质量提示词 | 交易抽成（20%） | 商业化模型不同，是交易平台非创作工具 | 活跃卖家数千 |
| FlowGPT | 提示词社区 + 分享平台，侧重 LLM 提示词 | 免费 + Premium | 社区导向，非视频专用；缺乏结构化编辑 | 社区活跃度高 |
| VideoPrompt.online | 视频提示词生成工具 | 免费 | 功能简单，无 AI 扩写；仅支持基础模板 | 早期产品 |
| AgentDock | 多 AI Agent 管理平台 | SaaS 订阅 | 通用 AI 工具平台，非视频提示词专用 | 新兴产品 |
| Sora 官方 Guide | OpenAI 提供的 Sora 提示词编写指南 | 免费文档 | 仅为指南文档，无工具化交互 | 官方权威 |
| 即梦平台内置编辑器 | Seedance 平台自带的提示词输入界面 | 平台内免费 | 锁定单一平台，无跨模型能力 | 平台原生工具 |

### 市场洞察

- **市场空白**：当前市场缺乏同时适配 Sora + Seedance 的专业视频提示词编辑器，现有工具要么是通用提示词平台（不针对视频），要么是单一平台内置工具（无跨模型能力）
- **差异化机会**：
  1. **双模型适配**：一次编写，输出 Sora + Seedance 两种格式——这是核心差异化壁垒
  2. **结构化 + AI 双驱动**：不仅提供模板，更通过结构化参数面板降低入门门槛、通过 AI 扩写提升输出质量
  3. **垂直聚焦**：专注视频提示词这一细分领域，而非泛化 AI 提示词工具
- **潜在风险**：
  1. Sora/Seedance 平台自身可能内置更强大的提示词编辑功能
  2. 视频生成 API 开放后，提示词工具的独立价值可能被进一步整合
  3. 新竞品可能快速跟进双模型适配功能

### SWOT 分析

| | 有利 | 不利 |
|---|------|------|
| **内部** | **S 优势**：双模型适配先发优势；结构化+AI 双重辅助降低门槛；Web App 轻量快速交付 | **W 劣势**：无用户数据验证画像；依赖第三方 LLM API（成本不可控）；团队规模小 |
| **外部** | **O 机会**：AI 视频生成市场 2026 年爆发；两大平台用户基数快速增长；提示词工具细分市场空白 | **T 威胁**：平台方内置提示词编辑功能；新竞品快速跟进；模型提示词格式变更频繁 |
